# Security

## Reporting a vulnerability

Use [GitHub's private vulnerability reporting](https://github.com/howdoiusekeyboard/react-haptics/security/advisories/new) to report security issues. Do not open a public issue.

You'll receive acknowledgment within 48 hours and a follow-up with next steps.

Vulnerabilities in dependencies should be reported upstream unless the issue is specific to how react-haptics uses that dependency.
